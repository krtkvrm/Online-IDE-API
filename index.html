Online IDE API

API Webservice to Run Programs for Online IDEs and Online Judges Edit

Documentation

Installation

  • Clone the Repository
    $ git clone https://github.com/vkartik97/Online-IDE-API
  • Install Dependencies
    $ npm install
  • Test for Compilers and Interpreters on the System
    $ npm test
  • Run the Application
    $ npm start

API

  • API is Called on localhost:8000/run by default, can be changned in server.js
  • API Request contains following data in JSON:
    • program: Source code of program in base64
    • lang: Programming Language of the Source Code
    • inputf: input to the program via STDIN
    • output: output to the program via STOUT
    Example :
    {
        "program": "aSA9IGludChpbnB1dCgpKQ0KDQpwcmludChpKzEsIGVuZD0nJyk=",
        "lang": "py3",
        "inputf": "1",
        "output": "2"
    }
  • API Response contains following data in JSON:
    • result: The Result of Execution with input and output.
      • AC: All Correct, input matches with output
      • WA: Wrong, input doesnt matche with output
      • ERR: Error in Program
    • error: The error message, if result is ERR
    • output: output of execuition
    Example for AC:
    { "result": "AC", "output": "2" }
    Example for WA:
    { "result": "WA", "output": "5" }
    Example for ERR:
